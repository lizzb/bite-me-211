    cd :    c hange  d irectory
    ls :   lists the files in the current directory

    change directory (cd) to where your github repository is stored in your files. If you're not sure what directory you are currently in, use ls to see what files are stored where you currently are.

    cd ~ :    c hange  d irectory to your user root folder (Where the folders for Applications, Desktop, Library, Downloads, etc. are located.) Most people keep their github repositories in a folder in this directory.
